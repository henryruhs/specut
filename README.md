Installation
------------

Install on your system:

```
npm install specut --global
```


Usage
-----

Run the command:

```
specut [options]

-V, --version
-C, --config <config>
-A, --amount <amount>
-M, --mode <mode>
-P, --path <path>
-h, --help
```
